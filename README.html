Face Detection Project README

Project Overview

The Face Detection project leverages advanced machine learning and computer vision techniques to accurately identify and track human faces in images and video streams. Utilizing a combination of TensorFlow, Keras, and OpenCV, this project offers a robust solution for applications requiring real-time face detection capabilities. For spacial complexity reason, the original and augmented dataset were left out of the git repository.

Key Features

  • Real-Time Detection: Detect faces in real-time with high accuracy in video streams using a pre-trained deep learning model.
  • Static Image Support: Apply face detection to static images, allowing for a wide range of image processing applications.
  • Advanced Machine Learning Models: Incorporates cutting-edge models like VGG16 for feature extraction, combined with custom layers for efficient face detection.
  • Data Augmentation: Implements data augmentation to enhance the diversity of training data and improve model robustness.
  • Easy-to-Use: Designed with simplicity in mind, the project includes user-friendly scripts for quick setup and deployment.

Installation Guide

  1. Clone the Repository: Start by cloning this repository to your local machine.
  2. Install Dependencies: Run the following command to install the necessary Python dependencies:
    pip install -r requirements.txt

How to Use

- For Real-Time Video: Launch face detection in real-time video streams with the face_detection_realtime.py script.
- For Static Images: Use the face_detection_image.py script to detect faces in individual images. Provide the image path as an argument.

Prerequisites

Ensure you have the following installed:

  • Python 3.6-3.9
  • TensorFlow 2.x
  • Keras
  • OpenCV
  • Albumentations (for data augmentation)
  • LabelMe (for image annotation)
